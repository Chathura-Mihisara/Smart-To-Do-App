# 📝 Smart To-Do App (Vanilla JS)

A modern and responsive To-Do application built using **Vanilla JavaScript**.

This app helps users manage daily tasks efficiently with features like adding, editing, deleting, and marking tasks as completed — all with persistent storage using **localStorage**.

---

## 🚀 Features

* ✅ Add new tasks
* ✏️ Edit existing tasks
* 🗑️ Delete tasks
* ✔️ Mark tasks as completed
* 💾 Data saved using localStorage
* 🎨 Smooth UI animations and hover effects
* 🧊 Custom scrollbar design
* 📱 Responsive and clean interface

---

## 🛠️ Tech Stack

* HTML5
* CSS3 (Flexbox, Animations)
* JavaScript (ES6+)
* LocalStorage API
